Keyboard Shortcuts

  • c - Toggle Control Mode
  • r - Reset Camera Position to Default
  • b - Reset Brain Position to Default
  • l - Toggle Labels
  • f - Toggle Flares
  • p - Toggle Particles
  • s - Toggle Slice Images if Available
  • d - Dump Camera and Controller to console
  • space - Deselect All
  • t - Trace Path, 1-0 to switch modes (only mode 1 currently available)
  • h - To close this help
